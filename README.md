# Tux Afterburner
 Overclocking and monitoring application for Nvidia and AMD

## Features

\* - done

? - experimental

! - work in progress / planned

- Core overclock (!)

- Memory overclock (!)

- Power limiting (!)

- FAN speed control (?)

- Monitoring (?)

- Apply on boot (!)

- Intel Arc Support (!)

## Nvidia

**GPU generations:**
    
- Full support: GTX 1070 and newer
- limited support: GTX 600, 700, 900 series and GTX 1050, 1060
- no support: GTX 640, GTX 500 and below, GT series

## AMD

**GPU generations:**
    
- Full support: RDNA1 series and newer
- limited support: RX 400, 500 series
- no support: GCN 3.0 and older
